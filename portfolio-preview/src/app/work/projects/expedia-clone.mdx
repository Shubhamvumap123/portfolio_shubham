## Overview

This project is a clone of the popular Expedia travel portal, focusing on providing a seamless user experience for browsing and booking travel services.

## Features

-   **Hotel Booking**: Browse and select hotels.
-   **Activities**: Explore various travel activities.
-   **Services**: Comprehensive travel service options.

## Tech Stack

-   HTML5
-   CSS3
-   JavaScript
