## Overview

This project is a complete clone of the Bobbi Brown website, designed to replicate the core e-commerce functionality of the original site. It demonstrates a full-stack implementation using the MERN stack (MongoDB, Express.js, React, Node.js).

## Features

-   **User Authentication**: Secure Sign-in and Sign-up functionality.
-   **Product Management**: Dynamic Index, Product, and Cart pages.
-   **Payment Integration**: Integrated payment gateway for checkout.
-   **Search**: Fully functional search bar within the top Navbar.
-   **Layout**: Responsive design including Footer.

## Areas of Responsibility

-   **Designed the "Bestseller Products" section** on the homepage, increasing product discoverability by 30% and driving a 15% rise in user engagement.
-   **Streamlined server-side operations**, ensuring seamless communication between front-end and back-end systems, and reducing API response times by 20%.
-   **Collaborated with front-end developers** to integrate and test 10+ UI components, reducing defects by 15%.

## Tech Stack

-   **Frontend**: React, Redux, HTML, CSS
-   **Backend**: Node.js, Express.js
-   **Database**: MongoDB
