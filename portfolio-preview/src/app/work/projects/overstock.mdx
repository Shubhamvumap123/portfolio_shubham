## Overview

Overstock is a comprehensive booking platform designed to simplify the reservation process for various services, including flights and accommodations.

## Features

-   **Booking System**: Streamlined interface for booking flights and flats.
-   **User Interface**: Clean and intuitive design using HTML and CSS.
-   **Responsive Layout**: Accessible on various devices.
