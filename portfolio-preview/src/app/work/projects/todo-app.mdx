## Overview

A robust task management application built with TypeScript, ensuring type safety and code maintainability.

## Features

-   **Task Creation**: Easily add new tasks.
-   **State Management**: Toggle between pending and completed states.
-   **Filtering**: View tasks by status.

## Tech Stack

-   TypeScript
-   HTML5
-   CSS3
